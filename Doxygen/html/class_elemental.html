Detailed Description

A class for creating clones of a specific type of monster (elemental mob).

Author:
Craig van Heerden U15029779, Dedre Olwage U15015239

Description

This class provides the functionality of creating clones of the mob type Elemental, as well as give the mob the ability to attack another player or take damage itself.
